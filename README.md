## 📖 是什么？

**鸿蒙代码工坊** 是一份覆盖 HarmonyOS 全流程开发的 AI 知识库，从 ArkTS 代码编写到 AppGallery 上架，一站式搞定。

### 与其他鸿蒙技能的区别

| 别人只做 | 我们还做 |
|----------|----------|
| 给 API 文档索引 | ✅ **确保代码能编译通过**（31 项自检清单） |
| 贴示例代码 | ✅ **告诉你为什么这样写、哪里会踩坑** |
| 覆盖 API 23 | ✅ **API 22~26 全版本，含 HarmonyOS 7** |
| 开发完不管上架 | ✅ **从签名打包到 AGC 审核全流程覆盖** |
| 文档搬运 | ✅ **来自真实项目的踩坑记录** |
| 让你自己找代码 | ✅ **30 个即用模板 + 5 个高级模式** |

## 🚀 快速使用

### 在 AI 工具中使用

本技能支持一键安装到以下 AI 工具：

```bash
# 克隆仓库
git clone https://github.com/FFF-119/harmonyos-code-workshop.git

# 安装到你的项目（让 AI 助手懂鸿蒙）
cd harmonyos-code-workshop
bash scripts/install.sh
```

### 支持的 AI 工具

| 工具 | 安装方式 |
|:---|:---|
| **Cursor** | `bash install.sh cursor` |
| **Trae（字节跳动）** | `bash install.sh trae` |
| **Claude Code** | `bash install.sh claude` |
| **GitHub Copilot** | `bash install.sh copilot` |
| **Windsurf** | `bash install.sh windsurf` |
| **ChatGPT** | 复制 `dist/chatgpt/system-prompt.txt` 到自定义指令 |
| **Gemini CLI** | `bash install.sh gemini` |
| **Continue.dev** | `bash install.sh continue` |

### 在 SkillHub 上使用

直接在 [SkillHub](https://skillhub.cn) 搜索「鸿蒙代码工坊」即可安装。
